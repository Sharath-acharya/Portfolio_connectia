# Sharath Acharya - Data Analyst Portfolio

A professional Data Analyst portfolio website built with React, TypeScript, and Tailwind CSS featuring stunning animations and a sleek black theme.

## Setup

1. Install dependencies:
```bash
npm install
```

2. **Add your profile photo:**
   - Save your photo as `profile.jpg` in the `public` folder
   - Or update the image path in `src/App.tsx` (line with `src="data:image/svg+xml...`)

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
